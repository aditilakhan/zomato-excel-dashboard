Zomato-Analysis

## Project Overview
This dashboard analyzes Zomato restaurant data using Microsoft Excel.

## Features
- Restaurant ratings analysis
- City-wise restaurant distribution
- Cuisine analysis
- Interactive slicers and charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
